好的配色是设计中最难的环节之一。对于个人博客这种需要长期陪伴自己的空间，配色更应该是安静、耐看、且富有个性的。

## 从插画到配色

这次的设计灵感来自一幅骑行插画。画面中有几个关键的色彩层次：

- **天空** — 蜜桃粉到淡橙的渐变，温暖而不喧闹
- **道路** — 深到接近藏青的蓝色，沉稳安静
- **草地** — 明亮但柔和的翠绿，代表生命力
- **太阳** — 浅黄色，作为高光点缀
- **远山** — 淡淡的粉紫色，增加层次感

## 色彩映射

将这些色彩提炼为设计系统的色板：

- **背景** `#FDFBF9` — 页面背景，奶油暖白
- **主标题** `#1B3A5C` — 文章标题、重要文字
- **正文** `#2D3748` — 正文段落
- **次要文字** `#718096` — 日期、辅助信息
- **链接** `#E8957C` — 链接、悬停状态
- **强调** `#F5C5B5` — 标签背景、悬停背景
- **辅助** `#7BAE7F` — 成功状态、辅助图标

## 在 Tailwind 中配置

在 `tailwind.config.ts` 中扩展自定义颜色：

```ts
theme: {
  extend: {
    colors: {
      cream: "#FDFBF9",
      "deep-blue": "#1B3A5C",
      "soft-coral": "#E8957C",
      "muted-coral": "#F5C5B5",
      "sage-green": "#7BAE7F",
      "text-primary": "#2D3748",
      "text-secondary": "#718096",
      "border-light": "#E8DDD8",
    },
  },
}
```

## 关于安静的设计

这套配色方案的核心原则是**安静**：

- 没有高饱和度的纯色
- 冷暖色调之间保持平衡
- 色彩对比度适中，不会刺眼
- 留白与色彩面积的比例保守

设计不是关于添加多少，而是关于**恰到好处**地保留什么。
